# frontend-playground
* UI를 프로토타이핑
* 하나의 주제 -> html 파일 1개로 모두 표현 (html + css + js)

Example:

빨간색 div를 만들기로 정했으면 red-div.html 파일 하나에 모두 구현해서 커밋


```html
<!DOCTYPE html>
<html>
<style>
    body {
        margin: 0;
        padding: 0;
    }

    .red {
        width: 500px;
        height: 500px;
        background-color: red;
    }
</style>
<head>
    <meta charset="UTF-8">
    <title>Red Div</title>
</head>
<body>

<div class="red"></div>

<script>
    // script
</script>
</body>
</html>
```
